# express-local-library

Local Library App using **server-side rendering** written in Express.

Install the dependencies.

```sh
cd express-local-library
npm install
```

Create **.env** file at the root of the project.
Set MONGODB_URI=**your_mongodb_uri** [(see)](https://www.mongodb.com/docs/guides/atlas/connection-string/)

> You can also set database name (e.g localLibrary) in uri. It is not mandatory.

```sh
npm start or npm run devstart
```

Check [localhost:3000](http://localhost:3000/).
